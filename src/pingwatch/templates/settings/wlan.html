WLAN-Ueberwachung

PingWatch misst Signal-Staerke + Re-Assoziationen via "iw dev wlan0 link" + nl80211. Wird benutzt um Aussetzer als WLAN-Problem (statt Netzwerk-Problem) zu klassifizieren.

Hinweis: Hier kannst du das WLAN wechseln. Ein Pi-Neustart ist beim WLAN-Wechsel nicht noetig — NetworkManager uebernimmt das live.

Interface Name des Wireless-Devices. Standard wlan0. Aendere nur wenn dein Pi mehrere Interfaces hat.
